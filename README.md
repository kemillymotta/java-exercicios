# Java Exercícios

Repositório criado para armazenar exercícios, exemplos e experimentos enquanto estudo Java.

A ideia deste repositório é registrar minha evolução na linguagem, praticando desde conceitos básicos até programação orientada a objetos.

---

# Estrutura do Repositório

Os códigos estão organizados por tipo de conteúdo.

📁 exercicios  
Exercícios propostos em aulas, cursos ou listas de prática.

📁 exemplosAulas  
Códigos feitos acompanhando as aulas, normalmente para testar conceitos específicos da linguagem.

📁 desafios  
Problemas um pouco maiores ou exercícios extras para praticar lógica e modelagem.

Dentro dessas pastas, os projetos geralmente seguem a estrutura padrão Java:

src  
 ┣ application → classe principal com o `main`  
 ┗ entities → classes do domínio do exercício

Essa separação ajuda a simular como projetos Java costumam ser organizados.
